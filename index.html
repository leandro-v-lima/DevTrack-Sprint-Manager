Como funciona

📄 Importação CSV
  1. Exporte os tickets da sua ferramenta de produto como CSV ou TXT
  2. Clique na área de upload ou arraste o arquivo
  3. O sistema mapeia as colunas automaticamente (fuzzy matching)
  4. Revise a prévia — duplicatas são destacadas em amarelo
  5. Confirme a importação — apenas registros novos são adicionados
🔌 Integração API
  1. Informe a URL base da sua API de tickets e a chave de autenticação
  2. Clique em "Testar Conexão" para validar as credenciais
  3. Use "Sincronizar Agora" para importar os tickets mais recentes
  4. Ative o polling automático para sincronização periódica
